Put your own legally-obtained .mp4 video files in THIS folder.

Then add them in the app's grown-up Settings screen:
  Type = "Local video file (offline)"
  File path = media/your-file-name.mp4

Local files are the most reliable choice for a flight — they play fully
offline with no algorithm, no ads, and no connection needed.

Keep file names simple (lowercase, dashes, no spaces), e.g.:
  media/puffin-rock-ep1.mp4
  media/bedtime-stories.mp4
